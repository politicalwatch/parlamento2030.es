PARLAMENTO2030.ES
=============

## Requirements

NodeJS > 8.0

## Setup

```
git clone git@github.com:ciecode/tipi-frontend.git
cd tipi-frontend
npm install
cp src/config/index.js.example src/config/index.js
```

Finally, edit *config/index.js* file with your specific values.


## Run

```
npm run serve
```
